# BKdV-S1 session log

- **Round 1** (E1/F1, kind=negative, is_trivial=false): Naive stack — Fourier pseudospectral + classical RK4 over the full RHS (v_xxx explicit) + no dealiasing, amp=1.5, dt=2e-4. Blew up before t=0.5 with overflow on v² / u·v then NaN through the FFT. Failure mode: broadband aliasing instability. Decision: add 2/3-rule dealias (single component change).
- **Round 2** (E2/F2, kind=positive, is_trivial=false): Added 2/3-rule dealias on every nonlinear product; everything else (RK4, dt=2e-4, amp=1.5) unchanged. Reached T=10. mass_v conserved to ~1e-12; sup ≈ 4; tail-band energy at machine zero. Aliasing identified as the dominant R1 failure mechanism; explicit-RK4-on-v_xxx is in fact stable at dt=2e-4 once the alias cutoff lowers effective k_max. Decision: hold the stack and push amp to the top of the range to look for a second failure mode.
- **Round 3** (E3/F3, kind=positive, is_trivial=false): Same solver as R2; amp raised 1.5 → 3.0. Reached T=10. mass_v exactly conserved; sup_u doubles to ~10.5 (Burgers-side steepening from 3v² source); sup_v decays from 3.0 → 1.16 (KdV soliton broadening / radiation); edge-band energy fraction stays < 1e-5. No second numerical failure mode triggered at this resolution. Working stack is amp-robust over [1.5, 3.0]. Decision: stop_useful; recommend the R2 stack for Stage-2 across the full required amp range.
